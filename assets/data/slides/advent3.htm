And so, onto the application

  • Sammy.Application is the base prototype for defining applications. When creating a sammy application we use $.sammy()
  • An application is a collection of routes and bound events that are attached to an element when .run() is called
  • Sammy.Application requires single argument app_function that is evaluated within the context of the application
  • Also takes and optional argument that is the element to which you want to attach the application
  • Within the context of the application this relates to the application itself
