Sammy.Application comes with a number of useful attributes and methods. The most useful of these are:

get, post, put, delete

Wrapper methods for the route method. These are the core of the Sammy.Application

$element

A method that returns a jQuery object of the element to which the application is bound

bind

creates a custom event that is bound to the current applications. Works well with...

trigger

triggers a custom event


...There are many more methods attached to the Application object see the sammy.js api documentation for more information
